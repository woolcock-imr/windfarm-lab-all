1-Back
This task compares the position of letters displayed on the screen.

Remember the position of the first letter.
Compare it to the position of all other letters.
If the position of the current letter matches the position of the first letter press M on the keyboard for Match as quickly as possible.
If the position of the current letter does not match press N for No Match as quickly as possible.

To begin, press the start button (A green dot will warn you that a letter is about to appear) This task starts IMMEDIATELY after pressing start
